# Use Case: Transfer Money

| **Field** | **Description** |
|------------|-----------------|
| **Actor** | User |
| **Trigger** | User selects "Transfer Money" |
| **Preconditions** | User is authenticated and has sufficient balance |
| **Main Flow** | 1. User inputs recipient and amount  <br>2. System validates recipient  <br>3. System checks balance  <br>4. System executes transfer  <br>5. System shows confirmation |
| **Postconditions** | Transaction record created, balances updated |
